Текстовая приключенческая игра, где игрок исследует таинственный лабиринт, решает загадки, собирает предметы и пытается найти легендарные сокровища.
## Установка

### Способ 1: через Poetry
```bash
# Установка зависимостей
poetry install

# Запуск игры
poetry run python labyrinth_game/main.py

### Способ 2: через Make

# Установка
make install

# Запуск
make project
